# TIKVA - Centro de Inteligência Robótica

## Sobre
Plataforma para seleção e deploy de modelos VLA para robótica industrial.

## Stack
- React 18 + TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Estética
- Dark mode luxury tech
- Cores: Verde escuro (#0A1F1C), Dourado (#D4AF37), Preto
- Fontes: Instrument Serif (display), Plus Jakarta Sans (body), JetBrains Mono (code)
- Animações suaves e sofisticadas

## Comandos
- `npm run dev` - desenvolvimento
- `npm run build` - produção

## Estrutura
src/
  components/    # Componentes reutilizáveis
  pages/         # Páginas principais
  hooks/         # Custom hooks
  utils/         # Utilitários
  assets/        # Imagens e vídeos
